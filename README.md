# Gushwork Assignment

This project is a **responsive web page implementation** based on the provided Figma design. The page is developed using **vanilla HTML, CSS, and JavaScript only**, without using any frameworks or external libraries, as required in the assignment instructions.

The goal of this project is to accurately translate the design into a fully functional, responsive webpage while maintaining clean code structure and best development practices.

---

# 🔗 Figma Design Reference

The implementation follows the design specifications from the Figma file:

👉 [Figma Design](https://www.figma.com/design/DOv07H7C2tA5UrVLhmfwfW/Gushwork-Assignment?node-id=490-8785)

All layout components, typography, spacing, colors, and visual elements have been implemented to match the design as closely as possible.

---

# 🚀 Features

## 1. Fully Responsive Layout

The webpage is designed to work seamlessly across multiple screen sizes:

* Desktop
* Tablet
* Mobile devices

Responsive behavior is implemented using **CSS media queries**, **Flexbox**, and **CSS Grid** to maintain layout consistency across different viewports.

---

## 2. Sticky Header Functionality

A dynamic sticky header is implemented using JavaScript.

### Behavior

* The header becomes visible when the user scrolls **beyond the first fold of the page**
* It positions itself **above the navigation bar**
* The header **disappears when scrolling back up**
* Smooth transitions and animations are used to improve the user experience

This ensures the navigation elements remain accessible while maintaining a clean UI.

---

## 3. Image Carousel with Zoom Feature

An interactive **image carousel** is implemented as shown in the Figma design.

### Functionality

* Users can navigate through carousel images
* Hovering over an image shows a **zoomed preview**
* Smooth hover effects and transitions are applied
* The carousel is responsive and works across different devices

The zoom functionality improves product visualization and enhances the overall user experience.
